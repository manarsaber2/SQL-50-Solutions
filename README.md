This repository contains my solutions to the **SQL 50** problems from LeetCode.

##  About SQL50
The SQL50 challenge is a curated list of 50 SQL problems on LeetCode designed to practice:
- SELECT statements
- JOINs
- Aggregations
- Window Functions
- Advanced queries
